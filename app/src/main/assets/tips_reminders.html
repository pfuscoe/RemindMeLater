Tips: Reminders

Tap the Add button in the menu bar to create a new Reminder. Tap a Reminder to edit its details.

Select Edit Categories from the drop-down menu to make changes to your reminder categories.

When you receive a reminder notification, you can:

  • Mark it Done to reset it based on its recurrence (or delete it if it is non-recurring)
  • Hit Snooze to push it back for one day
  • Hit Hibernate to push it back two weeks
  • Tap the notification body to edit its details

The User Settings option in the drop-down menu allows you to:

  • Edit your Display Name
  • Set the time of day to receive reminder notifications
  • Clear any unused reminder categories
